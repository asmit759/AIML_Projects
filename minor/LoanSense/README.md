# LoanSense — CreditWise Loan Approval System

## Overview

**LoanSense** is a Machine Learning–based creditwise loan approval prediction system designed to assist financial institutions in making faster, consistent, and data-driven lending decisions.

The system analyzes applicant information and predicts whether a loan should be **Approved** or **Rejected**, helping reduce manual effort and human bias in traditional loan verification processes.

---

## 🎯 Problem Statement

Financial institutions receive hundreds of loan applications daily. Traditional approval methods rely on manual verification of:

* Income proofs
* Employment status
* Credit history
* Financial documents

This manual process is:

* Time-consuming
* Inconsistent
* Prone to bias

LoanSense aims to automate preliminary loan evaluation using Machine Learning.

---

## 🧠 Machine Learning Approach

### Data Preprocessing

* Handling missing values
* One-Hot Encoding for categorical features
* Feature Scaling
* Removing irrelevant features

### Models Implemented

* Logistic Regression
* KNN
* Gausian NB
* Decision Tree Classifier

### Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
